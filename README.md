# Vinoteca – Sistema de Gestión de Pedidos (Demo Estática)

Proyecto académico para **Taller de Proyectos II – Módulo 3 (A3)**.  
Esta versión **estática** (sin backend) permite demostrar el flujo funcional y el avance ≥60%.

## Funcionalidades
- Alta de pedidos (cliente, producto, cantidad, precio).
- Cálculo de total automático.
- Listado con búsqueda y orden por columnas.
- Persistencia local con `localStorage`.
- Exportación a CSV.

## Ver la demo
- Abrí `index.html` (doble clic), o
- Subí este repo a GitHub y activá **GitHub Pages** (Settings → Pages → Deploy from a branch → main / root → Save).
